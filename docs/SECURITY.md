## Reporting a Vulnerability

### Process

1. **DO NOT** create a public issue
2. Email security@superpassword.app
3. Include details:
   - Description
   - Steps to reproduce
   - Impact assessment
   - Possible fixes

### Response Timeline

- Initial response: 24 hours
- Assessment: 72 hours
- Fix timeline: case-by-case
- Public disclosure: coordinated
